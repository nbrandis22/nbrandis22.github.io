                    Fit Tracker

For our CS35L final project, my group decided to create a fitness tracker web application. The app allows users to log workout information and visit old workouts, as well as observe progress over time and copy the structure of previous workouts. Our motivation was that we all enjoy working out, but none of us had anything to use to keep track of our workouts each day. FitTracker is an attempt to solve this issue.

A screenshot of a "Workout" page of Fit Tracker.

To implement this app, we utilized React for the frontend and Node and Firebase for the backend. For the login page, we used React’s Google login feature to allow users to sign in with an existing Google account. We used Material UI to design the components of the front end, such as the exercise cards and the navigation bar. To route between different pages in the app (i.e. login, workout, and history), we used React Router. Then for the backend, we saved user data into the Google Firebase realtime database. To communicate between the frontend and backend, we use Axios for HTTP get requests that would request data at a specific URL from the Node server.

If you wish to download the project and run it for yourself, you can click the "Project Download" link below to download the code. From there, ensure you have the node and npm modules installed on your computer. Once you do, open the '35lfitttracker' folder on the command line and cd into the 'src' directory. Then, run 'node server.mjs' to start up the Node server. Finally, open a new terminal session, and in the same 'src' directory, run the command 'npm start'. This should open up the React application in your browser, and you should be able to play with the app yourself.
